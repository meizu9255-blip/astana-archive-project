# Астана қаласы Мемлекеттік архивінің ақпараттық жүйесі (Веб-портал)

## Қысқаша сипаттама

Бұл дипломдық жобаның басты мақсаты — Астана қаласының мемлекеттік архиві үшін азаматтарға қызмет көрсету процестерін толығымен цифрландыратын заманауи ақпараттық жүйені әзірлеу және енгізу. Веб-портал халықпен өзара әрекеттесуді жеңілдетіп, мемлекеттік қызметтердің қолжетімділігі мен ашықтығын арттырады.

Жүйенің негізгі функционалдық мүмкіндіктері:
* **Азаматтардан өтініш қабылдау жүйесі:** Онлайн форматта мұрағаттық, генеалогиялық және әлеуметтік-құқықтық анықтамаларға тапсырыс қабылдау.
* **Paperless (қағазсыз) технологиясы:** Әкімшілік панель арқылы PDF форматындағы ресми мұрағаттық анықтамаларды (двутілді, мемлекеттік стандартқа сай) автоматты түрде генерациялау және азаматтарға тікелей порталдан жүктеп алу мүмкіндігін ұсыну.
* **AI чат-бот (Әмбебап Ассистент):** Азаматтарға тәулік бойы (24/7) кеңес беретін, сұраныстарды жылдам ресімдеуге көмектесетін жасанды интеллект негізіндегі виртуалды көмекші.

## Технологиялық стек

Жоба заманауи және жоғары өнімді веб-технологиялар негізінде жасалған:

* **Frontend:** React 18, Vite, Tailwind CSS (Пайдаланушы интерфейсін құру және бейімделгіш дизайн).
* **Backend:** Vercel Serverless Functions (REST API архитектурасын жүзеге асыру).
* **Database (Дерекқор):** Neon PostgreSQL (Деректерді реляциялық құрылымда қауіпсіз сақтау және басқару).
* **File Storage:** Vercel Blob Storage (Генерацияланған PDF құжаттарды бұлттық ортада сақтау).
* **AI Integration:** Google Gemini 1.5 Flash (Ассистенттің табиғи тілді өңдеу алгоритмдері).
* **PDF Generation:** html2canvas, jsPDF (Браузерлік қозғалтқыш арқылы жоғары сапалы құжаттарды жасау).

## Жобаны іске қосу (Local Setup)

Жобаны жергілікті ортада (local machine) іске қосу және тестілеу үшін төмендегі нұсқаулықты орындаңыз:

### 1. Репозиторийді клондау
Алдымен, жобаның бастапқы кодын өз компьютеріңізге жүктеп алыңыз:
```bash
git clone https://github.com/meizu9255-blip/astana-archive-project.git
cd astana-archive-project
```

### 2. Тәуелділіктерді орнату
Барлық қажетті NPM пакеттерін орнату үшін келесі бұйрықты орындаңыз:
```bash
npm install
```

### 3. Қоршаған орта айнымалыларын баптау (.env)
Жобаның түбірлік каталогында `.env` файлын құрып, келесі кілттерді толтырыңыз (бұл деректер серверлік функциялар мен дерекқорға қосылу үшін қажет):

```env
# PostgreSQL дерекқорының қосылу сілтемесі (Neon)
DATABASE_URL="postgres://user:password@ep-withered-haze-123456.eu-central-1.aws.neon.tech/neondb"

# Google Gemini жасанды интеллектінің API кілті
GEMINI_API_KEY="AIzaSyYourGeminiKeyHere..."

# Vercel Blob сақтау орнының токені
BLOB_READ_WRITE_TOKEN="vercel_blob_rw_YourTokenHere..."

# Әкімшілік панельге кіру үшін орнатылатын құпия сөз
ADMIN_PASSWORD="your_secure_password"
```

### 4. Жобаны іске қосу
Барлық баптаулар аяқталғаннан кейін, локальді серверді іске қосыңыз:
```bash
npm run dev
```
Жүйе әдетте `http://localhost:5173` мекенжайы бойынша қолжетімді болады.
